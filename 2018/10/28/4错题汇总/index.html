4错题汇总

发表于 2018-10-28

  1. 在命令行里书写命令时，大小写无所谓，比如 cd ..和 CD .. 没区别。

    答：×

  2. 命令行如何进入上级目录。

    答：cd .. 或cd ../

  3. 命令行如何进入当前目录中的a目录。

    答：cd a或cd ./a

  4. 目前我在/a/b目录，输入 cd /c/ 我会进入哪个目录？

    答： /c/

  5. 命令行如何在当前目录创建一个目录 xxx？答： mkdir xxx mkdir ./xxx mkdir -p "xxx"

  6. 目前我在/a/，我想创建 /a/b/c/d/e/目录，用一个 mkdir命令如何实现？答：mkdir -p /a/b/c/d/e/

  7. 如何删除目录 /a/xxx/ ? 答：rm -rf /a/xxx/

  8. 目前我在 /tmp/目录，运行 git clone git@github.com:FrankFang/nodejs-test.git,请问 /tmp/ 目录里会多出一个什么目录？ 答：nodejs-test

  9. 将 xxx 文件变动加入 git 暂存区的命令是什么？ 答：git add xxx

  10. 将 xxx文件变动提交到 git 本地仓库的命令是什么？ 答:git commit xxx

  11. git push 之前最好 git pull 一下? 答：√ , 为了将远程仓库拉到本地仓库，使得远程仓库和本地仓库保持相同。

  12. git 的默认远程仓库名叫做什么？ 答：origin

  13. git 的默认分支叫做什么？ 答：master

  14. git课外资料： http://www.runoob.com/git/git-tutorial.html

超级容易错的-教程里配置git的五大步骤：

git config --global user.name liaolinyan2018

git config --global user.email linyanliao@126.com

git config --global core.quotepath false

git config --global core.editor "vim"

git config --global push.default simple

说明：前两步是修改过的结果，每个人都不一样
