使用git在本地创建一个.git仓库

  1. git init 初始化空仓库
    首先在你常用的地方创建一个目录作为项目目录（不要放在桌面）: mkdir git-demo-1
    进入你创建的这个目录（一定要进入）:cd git-demo-1
    在git-demo-1目录下创建一个名为.git仓库:git init
    这时.git仓库是空仓库,但是它的结构很复杂。
    我们可以往git-demo-1目录里添加或者删除文件,如:touch index.htmlthenmkdir cssthen touch css/style.css

  2. git add 提交变动到暂存区
    通过第一步的操作,我们改变了当前文件夹git-demo-1的信息
    用git add 将当前目录（.表示当前目录）的变动信息 加到 暂存区里:git add .

  3. git commit 将暂存区变动提交到本地仓库
    使用 git commit -m "注释信息" 将你add过的内容「正式提交」到本地仓库（.git就是本地仓库），并添加一些注释信息，方便日后查阅: git commit . -m “这次变动了什么内容”
