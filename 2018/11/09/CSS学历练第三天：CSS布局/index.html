CSS学历练第三天：CSS布局

发表于 2018-11-09

CSS布局技巧

左右布局

三步走：在父元素上添加类 class="clearfix",在子元素上加 float:left(right);,在CSS样式表中加入:

.clearfix::after{
        content:'';
        display:block;
        clear:both;
       }

左中右布局

五步走：父元素有三个子元素，同理给父元素添加类 class="clearfix",在子元素上加 float:left(right);,在CSS样式表中加入:

.clearfix::after{
        content:'';
        display:block;
        clear:both;
       }

然后给后两个子元素设置margin-left,给父元素设置水平居中。

水平居中

<div class="father">  
  <div class="child">
    实现子元素水平和垂直居中
  </div>
</div>

水平居中在布局中经常使用。要考虑：子元素为行内元素还是块状元素，宽度一定还是宽度未定，采取的布局方案不同。下面进行分析：

行内元素居中:对父元素设置text-align:center;

定宽块状元素居中：设置左右margin值为auto,即margin:**px auto;

不定宽块级元素居中：方法一变成内联元素，设置子元素为display:inline,然后在父元素上设置text-align:center。方法二，定宽。

通用方案：flex布局，对父元素设置display:flex;justify-content:center;

垂直居中

垂直居中对于 子元素 是单行内联文本、多行内联文本以及块状元素采用的方案是不同的。

父元素高度确定，子元素为单行内联文本：设置父元素的height等于行高line-height

父元素高度确定，子元素为多行内联文本：设置父元素的display:table-cell或inline-block，再设置vertical-align:middle;

块状元素:设置子元素position:fixed（absolute），然后设置margin:auto;

通用方案: flex布局，给父元素设置display:flex; align-items:center;

    小细节的知识点：

    1.空标签就是没有儿子的标签，比如input就是空标签，button不是空标签。
  1. 伪元素 不是元素，可以看做一个span伪元素的用法-给所有的div元素之前或之后 加 内容。给伪元素加display:block;后，独占一行，我们都可以加border测试。
  3. 伪元素用法-before 和after都可以代替div，但是局限是before里不能再有before。例如用before画阴阳

    使用伪元素以后就可以用:before和:after代替div。然后我们只使用一个div就画出了阴阳图:before和:after是写在CSS样式里面的。

  4. 伪元素很装逼

  5. 工具 线性渐变代码生成器 ,写CSS就是会很多工具，不然有时候=代码都自己写就很麻烦。


  6. 给阴阳图加上CSS动画,阴阳转起来：查MDN。

    创建动画序列，需要使用animation属性或其子属性，该属性允许配置动画时间、时长以及其他动画细节，但该属性不能配置动画的实际表现，动画的实际表现是由 @keyframes规则实现。


  7. CSS有很多杂碎的东西，要多去试就能学到更多的东西。
  8. 对页面某一块，先实现功能内容，再加CSS样式，即先html，后CSS。
  9. a 是内联元素。内联元素的水平居中要写在父元素上面，子元素会继承父元素内容居中属性如<p style="text-align"><a>内联元素实现居中方法</a></p>。


  10. 从里往外写CSS样式，最后加padding,margin撑起来。
  11. 伪类是一个分号，如:hover :nth-child,它是不存在的，表示标签的某种状态。伪元素是两个分号，如： ::before ::after,伪元素是真实存在的，但是不是一个元素，它一个替代一个元素的作用。
  12. CSS样式复用 选择器,选择器{CSS样式}
  13. id是唯一标识一个标签的，class是一个大类。
  14. 状态机，js切换左中右三种状态,新学知识。






  15. display:inline-block;
    vertical-align: top;是配套使用的，设置了inline-block，就要同时设置后者。inlne-block可以让标签宽度等于文档流元素，另起一行。
  16. inlne-block既可以另起一行，又可以让标签宽度等于文档流元素，是收缩的。块级元素会继承父亲的属性。故可以利用文档流使大家的宽度都合适。
