cnode社区实战

发表于 2019-08-28 | 更新于: 2019-08-28 | | 阅读次数:
字数统计: 445 字 | 阅读时长 ≈ 2 分钟

cnode社区 项目主要是依赖cnode社区提供的API实现的。

使用vue 开发项目，就是写一个个组件，将组件”堆在一起”。就是应用了。

1 cnode社区的基本构架

该项目包括组件：

  • Header 头部
  • PostList 列表
    • Article 文章的详情页
      • SlideBar 侧边栏
    • UserInfo 用户个人信息
  • Psgination 分页组件

2 开始实战

2.1 Header组件

  • HTML ： img ul li div
  • CSS样式 和 布局 ：浮动布局&清除浮动 | padding垂直居中 | max-width+margin：0 auto 水平居中
  • JS ： 无

2.2 PostList组件

  • vue技术

    axios

API接口：https://cnodejs.org/api/v1/topics 获取帖子列表
可以加参数 limit page
拿到的参数分析 res.data[i].xxx

头像：author.avatar_url
回复量/浏览量 ：reply_count/visit_count
帖子的标题：title
需要对时间使用过滤器：
时间：last_reply_at
帖子分类：top: 代表是否置顶，绿色强调
good: 代表是否精华，绿色强调
tab 参数是表示除了置顶和精华之外的其余分区
–­share 分享
­–ask 问答
­–job 招聘

注： 帖子分类只能有一个

2.3 Article组件

API https://cnodejs.org/api/v1/topic/ + 帖子ID
